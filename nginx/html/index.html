Public gateway for content on IPFS

Fetch content by CID or IPNS. A secured API and WebUI are available for administrative actions.

Open Sample API Version

Quick endpoints

  • /ipfs/<CID> serve content
  • /ipns/<NAME> IPNS name resolution
  • /api/v0/version node version (auth)
  • /api/v0/add upload file (auth)

Upload file via API

Use Basic Auth from an allowlisted IP.

curl -u user:pass -F file=@/path/file https://ipfs.peaq.xyz/api/v0/add

Show in WebUI (restricted)

Copy CID into MFS to appear in Files.

curl -u user:pass -X POST "https://ipfs.peaq.xyz/api/v0/files/cp" \ --data-urlencode "arg=/ipfs/<CID>" --data-urlencode "arg=/uploads/name"

Pin content

Ensure persistence on this node.

curl -u user:pass -X POST "https://ipfs.peaq.xyz/api/v0/pin/add?arg=<CID>"
